# superpowers-lite skills

When the user asks about any of these topics, read the matching SKILL.md
before responding:

- Debugging                        → `<SKILLS_PATH>/systematic-debugging/SKILL.md`
- Brainstorming features           → `<SKILLS_PATH>/brainstorming/SKILL.md`
- Writing implementation plans     → `<SKILLS_PATH>/writing-plans/SKILL.md`
- Executing plans                  → `<SKILLS_PATH>/executing-plans/SKILL.md`
- Subagent-driven implementation   → `<SKILLS_PATH>/subagent-driven-development/SKILL.md`
- Parallel independent tasks       → `<SKILLS_PATH>/dispatching-parallel-agents/SKILL.md`

Tool-name mapping (Claude Code tool names → Gemini equivalents):
`<SKILLS_PATH>/using-superpowers/references/gemini-tools.md`

Always start by reading `<SKILLS_PATH>/using-superpowers/SKILL.md`.
